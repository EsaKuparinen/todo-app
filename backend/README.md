# Todo App Backend

## Description

[Nest](https://github.com/nestjs/nest) framework based backend for the ToDo App.

## Project setup

```bash
$ npm i
```

## Running the project

```bash
# development (with watch mode)
$ npm start
```

## Running tests

```bash
# unit tests
$ npm run test

# unit tests (with watch mode)
$ npm run test:watch
```
